データインポート手順

データの準備から取り込みまでの手順を説明します。

1. 初期データ準備
2. スクリプト実行
3. データインポート

1初期データを準備する

データは以下の形式でExcelで作成し、CSVファイルとして保存します。

データの必須フォーマット
A列 B列 C列 D列以降
郵便番号（必須） 住所（必須） 名称（必須） 任意の追加項目
重要:
  • 郵便番号は必ず含めてください（座標取得に使用します）
  • 郵便番号は「123-4567」または「1234567」の形式で記載してください
  • 住所は都道府県から記載してください
データ例：
100-0013,東京都千代田区霞が関1-2-1,中央省庁ビル,官公庁
530-0001,大阪府大阪市北区梅田1-1-3,大阪駅前ビル,商業施設
060-0001,北海道札幌市中央区北1条西2丁目,札幌中心ビル,オフィス
手順:
  1. ExcelなどでデータをA列、B列、C列の形式で作成する
  2. 「名前を付けて保存」で「CSV (カンマ区切り)」を選択
  3. ファイル名を「input.csv」として保存
  4. geocode_postal.pyと同じフォルダに配置

2スクリプトを実行する

準備したCSVデータに対してPythonスクリプトを実行し、座標を取得します。

1. スクリプトファイルの準備

以下のリンクからPythonスクリプトをダウンロードし、「geocode_postal.py」として保存してください。

Pythonスクリプトをダウンロード
注意: ファイルをダウンロードした後、必ずファイル名が「geocode_postal.py」になっていることを確認してください。ブラウザによっては自動的に「.txt」拡張子が付く場合があります。その場合は拡張子を削除してください。
2. 必要な準備（Pythonのインストール）

スクリプトを実行するには、Pythonとライブラリのインストールが必要です。

  1. Python公式サイトからWindows用のインストーラーをダウンロードします
  2. インストーラーを実行し、「Add Python to PATH」にチェックを入れてインストールします
  3. インストール後、コマンドプロンプトを開いて以下のコマンドで必要なライブラリをインストールします：
    pip install pandas requests
  4. 正常にインストールされたか確認するには、以下のコマンドを実行します：
    python --version
    pip list
  1. MacにはPythonがプリインストールされていますが、最新版が推奨されます
  2. 最新版をインストールするには、Homebrewを使用するのが簡単です：
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. Homebrewでの Python インストール：
    brew install python
  4. 必要なライブラリのインストール：
    pip3 install pandas requests
  5. 正常にインストールされたか確認：
    python3 --version
    pip3 list
  1. ほとんどのLinuxディストリビューションにはPythonが標準でインストールされています
  2. 最新のPythonをインストールするには、パッケージマネージャを使用します（Ubuntu/Debianの場合）：
    sudo apt update sudo apt install python3 python3-pip
  3. 必要なライブラリのインストール：
    pip3 install pandas requests
  4. 正常にインストールされたか確認：
    python3 --version
    pip3 list
3. スクリプト実行
重要: スクリプトを実行するには、「input.csv」と「geocode_postal.py」が保存されている同じフォルダで実行する必要があります。
【手順1】ファイルの準備
  1. 「geocode_postal.py」をダウンロードして保存した場所を確認します
  2. 「input.csv」ファイルを同じフォルダに作成・保存してください
【手順2】コマンドラインでフォルダに移動
  1. コマンドプロンプトを開きます (Windowsキー + R を押し、「cmd」と入力)
  2. 以下のコマンドでファイルのあるフォルダに移動します：
    cd C:\path\to\your\folder
    ※「C:\path\to\your\folder」は実際のファイル保存場所に置き換えてください
  3. 「dir」コマンドでファイル一覧を確認できます：
    dir
    入力したコマンドで「geocode_postal.py」と「input.csv」が表示されていることを確認してください
  1. ターミナルアプリを開きます
  2. 以下のコマンドでファイルのあるフォルダに移動します：
    cd /path/to/your/folder
    ※「/path/to/your/folder」は実際のファイル保存場所に置き換えてください
  3. 「ls」コマンドでファイル一覧を確認できます：
    ls
    入力したコマンドで「geocode_postal.py」と「input.csv」が表示されていることを確認してください
【手順3】Pythonスクリプトの実行

フォルダに移動できたら、以下のコマンドを実行してスクリプトを起動します：

python geocode_postal.py

スクリプトが実行されると、郵便番号から座標を取得し、進捗状況が表示されます。処理が完了すると同じフォルダに output.csv ファイルが生成されます。

注意:
  • APIには利用制限があるため、大量のデータ処理には時間がかかる場合があります。
  • 「python」コマンドでエラーが出る場合は「python3」を試してください：
    python3 geocode_postal.py

3結果を確認してインポートする

スクリプト実行後に生成されたCSVファイルを確認し、アプリケーションにインポートします。

出力ファイル:

スクリプト実行後、同じフォルダに output.csv が生成されます。

データ確認:
  1. Excelなどでoutput.csvを開く
  2. 緯度(latitude)と経度(longitude)の列に値が正しく入っているか確認する
  3. データに問題がなければ、このファイルを使用してインポートします
データインポート手順:
  1. アプリケーション上部の「データインポート」ボタンをクリック
  2. 「基準データ名」と「対象データ名」を入力（例：「ホテル」「店舗」など）
  3. CSVファイルの内容を直接コピーして対応する入力欄に貼り付け：
    • 基準データとして使用するデータ → 「基準データ（テキスト）」欄
    • 対象データとして使用するデータ → 「対象データ（テキスト）」欄
  4. 「データをインポート」ボタンをクリック
